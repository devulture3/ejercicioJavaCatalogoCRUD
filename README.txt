PARA EJECUTAR LA APLICACION ES NECESARTIO EJECUTARLA DESDE \scr\interfaz\VistaPrincipal.java

Notas:
Es necesario hacer la conexion a la base de datos para ejecutar la aplicacion, de lo contrario
mandara el mensaje "No se conecto a DB: {NOMBRE DE LA BASE DE DATOS}" y la aplicacion no iniciara
Las configuraciones necesarias para la base de datos se encuentran en el archivo: \scr\pruebatecnica\Conexion.java

Para la base de datos crear una base de datos nueva llamada bdpruebatec
importar a la base de datos el archivo bdpruebatec.sql
